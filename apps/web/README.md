# `@wildfire/web`

## Getting Started

```sh
bun run dev
```

### Adding `shadcn/ui` Components

> [!NOTE]
> Make sure you're in the `/apps/web` folder before running these commands.

```sh
bun run shadcn:add button
```

### Ports Used

Here are the ports you'll see in use locally:

- `5173`: WILDFIRE's Vite server
