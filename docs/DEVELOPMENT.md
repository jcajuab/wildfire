# DEVELOPMENT.md

## Prerequisites

- [Bun](https://bun.sh/docs/installation)
- [Docker](https://docs.docker.com/engine/install/)
  - [Docker Compose](https://docs.docker.com/compose/install/)
- [Visual Studio Code](https://code.visualstudio.com/Download) – Optional, but recommended.

## Getting Started

### Clone the Project

```sh
git clone https://github.com/jcajuab/wildfire.git
```

### Start Developing

```sh
cd wildfire/
bun install # Install dependencies
bun run dev # Starts development servers
```

## Project Structure

```
wildfire/
├── dist/        # Build output (generated by `bun run build`)
├── docs/        # Project documentations
├── packages/
│   ├── client/  # Frontend
│   ├── server/  # Backend
│   └── shared/  # Shared utilities and types
└── package.json
```
